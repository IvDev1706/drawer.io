# Drawer.io
Drawer.io es un editor grafico similar a ms paint con la peculiaridad
de que este es web, desarrollado principalmente con typescript y originalmente
proyecto de la materia de graficacion por computadoras.

## Graficas
El editor grafico posee caractristicas basicas en la parte de dibujo y
edicion, las herramientas graficas que se pueden utilizar estan:
- trazo de pincel.
- texto.
- linea recta.
- rectangulo normal y redondeado.
- triangulo equilatero.
- rombo o diamante.
- pentagono.
- hexagono.
- circulo.
- ovalo.

## Transformacions
Ademas de que se permiten las siguientes operaciones sobre las graficas:
- traslacion: seleccionar una grafica y moverla.
- escalado: cambiar el tamaño.
- rotacion: rotar la grafica n grados con respecto a su origen.
**Nota**: no todas las graficas pueden rotarse (circulo y ovalo) ya que el efecto es casi nulo, tampoco se puede aplicar a trazos de pincel.

## #Beta Exportar imagen
El editor permite exportar el lienzo a una imagen jpg o png para descargar
en el dispositivo local, esta caracteristica aun esta pendiente a su imple
mentacion, durante el proyecto original no se concidero y es un añadido de
la nueva version.

## ¿Porque web?
El porqur realizar esta aplicacion de forma web, es debido a la versatilidad
de acceso ya que al ser ejecutada en el navegador cualquier dispositivo puede
ejecutarla, por lo que la app puede ser usada en pc o dispositivo movil.
